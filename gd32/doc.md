## 使用说明

产品实现捕获外部输入的维持100ms-500ms的高电平来触发电源复位


### modbus-rtu 寄存器说明

允许通过modbus-rtu(rs485), 波特率9600，slave id为1，控制部分参数的设置， 支持保持寄存器操作，寄存器描述如下

|   保持寄存器 | 偏移 |   描述 | 默认值 |
| ---------- | ------ | -------- | ------ |
| 40001 |   0x00 |   打开电源的电平，1：高, 0: 低 | 0 |
| 40002 |   0x01 |   默认上电状态， 1： 上电， 0： 不上电    | 1 |
| 40003 |   0x02 |   复位前延时时间(ms)   | 3000 |
| 40004 |   0x03 |   复位时关闭电源和打开电源之间的时间差(ms) | 3000 |
| 40005 |   0x04 |   通过寄存器复位，写 1 复位，其他无效   | - |
| 40006 |   0x05 |   硬件重启， 写1 重启，其他无效 | - |

复位操作：寄存器40001为1时，先拉低后拉高完成复位操作, 寄存器40001为0时，先拉高后拉低完成复位操作。触发复位时先延时40003寄存器值(ms)再复位，复位间隔时间是40004寄存器的值(ms)

开机时，寄存器40002为1时电源设置为40001寄存器的电平， 为0时设置40001相反的电平。

寄存器40001 - 40004 的值会实时保存到flash中，重启设备会从flash读取到寄存器中。
